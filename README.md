# Study Strategy

## Phase Strategy

1.  Start Python and Probability(then Statistics) together.
2.  Add NumPy and Pandas after Python basics + Linear Algebra.
3.  Learn SQL once comfortable with data handling.
4.  Begin Machine Learning after completing foundations.

------------------------------------------------------------------------


### Maximum: Two subjects at the same time.

# End Goal

After finishing this roadmap you should be able to:

-   Write Python for data analysis
-   Manipulate datasets using Pandas
-   Query databases with SQL
-   Understand probability and statistics foundations
-   Implement machine learning models
